## 📋 How to Print to Word

### Step 1: Generate the Diagram
1. Go to: **https://www.plantuml.com/plantuml/uml/**
2. Copy all code from `ERD_PLANTUML_PRINT_FRIENDLY.puml`
3. Paste into the editor
4. Click "Submit"

### Step 2: Export as PNG
1. Right-click on the diagram
2. Select "Save image as..."
3. Save as PNG file (e.g., `ERD.png`)

### Step 3: Insert into Word
1. Open Microsoft Word
2. Go to **Insert** → **Pictures** → **This Device**
3. Select your `ERD.png` file
4. Click **Insert**

### Step 4: Resize in Word
1. Click on the image
2. Drag corners to resize (keep aspect ratio)
3. Right-click → **Wrap Text** → **Square** (for better placement)

### Step 5: Print
1. Go to **File** → **Print**
2. Select your printer
3. Choose **Landscape** orientation (recommended)
4. Click **Print**

## 💡 Tips for Better Printing

### Tip 1: Landscape Orientation
- Gives more horizontal space
- Better for wide diagrams
- Recommended for ERD

### Tip 2: Fit to Page
- In Word: **File** → **Print** → **Fit to Page**
- Automatically scales to fit

### Tip 3: High Quality
- Export as PNG (not JPG)
- Use 300 DPI for printing
- Ensures text is readable

### Tip 4: Test Print
- Print to PDF first
- Check readability
- Adjust size if needed

## ❌ Troubleshooting

### Text Too Small
- Increase zoom in Word
- Or export at higher resolution

### Diagram Too Wide
- Use landscape orientation
- Reduce margins in Word
- Fit to page option

### Text Not Readable
- Check print quality setting
- Use color printer
- Increase DPI to 300

### Missing Tables
- Make sure you copied entire code
- Check PlantUML editor for errors
- Regenerate diagram
