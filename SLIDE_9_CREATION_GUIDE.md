# Slide 9 — User Interface Preview: Step-by-Step Creation Guide

## 📋 Overview
This guide shows you exactly how to create Slide 9 in PowerPoint with UI mockups and design system information.

---

## 🎯 Slide 9 Structure

Your slide should have:
1. **Title**: "User Interface Preview"
2. **Subtitle**: "Screenshots & Design System"
3. **Main Content**: UI mockups or screenshots
4. **Supporting Content**: Design system information

---

## 📝 Step 1: Create the Slide Title

### In PowerPoint:
1. Click **Insert** → **New Slide**
2. Choose **Title and Content** layout
3. Click on the title placeholder
4. Type: **"Slide 9 — User Interface Preview"**
5. Format:
   - Font: Arial or Calibri
   - Size: 44pt
   - Bold: Yes
   - Color: Dark blue (#1976D2)

---

## 🎨 Step 2: Add Subtitle

1. Click below the title
2. Add a text box with: **"Screenshots • Design System • Responsive Views"**
3. Format:
   - Font: Arial
   - Size: 20pt
   - Color: Gray (#757575)

---

## 📸 Step 3: Add UI Screenshots

### Option A: Use Actual Screenshots (Best)

1. **Take screenshots** from the running application:
   - Login page
   - Patient dashboard
   - Booking page
   - Calendar view
   - Medical vault
   - Dentist dashboard
   - Staff dashboard

2. **Insert screenshots**:
   - Click **Insert** → **Pictures** → **This Device**
   - Select screenshot files
   - Resize to fit slide

3. **Arrange in grid**:
   - 2 columns × 3 rows (6 screenshots)
   - Or 3 columns × 2 rows

4. **Add captions**:
   - Click **Insert** → **Text Box**
   - Add label below each screenshot
   - Format: Arial, 10pt, bold

---

### Option B: Create Mockups (If no screenshots available)

1. **Create Login Page Mockup**:
   - Click **Insert** → **Shapes** → **Rectangle**
   - Draw a rectangle (3" × 4")
   - Add text inside:
     ```
     CODE SMILES
     Dental Booking System
     
     Select User Type:
     ○ Patient  ○ Staff  ○ Dentist
     
     Email: [_____________]
     Password: [_____________]
     
     [LOGIN]  [FORGOT PASSWORD]
     ```
   - Format: Light gray background, blue text

2. **Create Patient Dashboard Mockup**:
   - Draw another rectangle
   - Add text:
     ```
     PATIENT DASHBOARD
     
     Welcome, John!
     
     Upcoming: 3
     Completed: 12
     
     Upcoming Appointments:
     • May 28, 10:00 AM - Cleaning
     • Jun 5, 2:30 PM - Root Canal
     
     [BOOK NEW APPOINTMENT]
     ```

3. **Create Booking Page Mockup**:
   - Draw another rectangle
   - Add text:
     ```
     BOOK AN APPOINTMENT
     
     Step 1: Select Service
     ☐ Cleaning - $50
     ☑ Root Canal - $200
     
     Step 2: Select Date & Time
     Date: [May 28, 2026]
     Times: ○ 9:00 AM  ○ 10:00 AM
     
     [SUBMIT]  [CANCEL]
     ```

## 🎯 Step 4: Add Design System Section

### Create Design System Box

1. Click **Insert** → **Text Box**
2. Add the following content:

```
DESIGN SYSTEM

Color Palette:
• Primary Blue: #1976D2
• Success Green: #4CAF50
• Warning Orange: #FF9800
• Error Red: #F44336
• Neutral Gray: #757575

Typography:
• Headers: Roboto Bold, 24px
• Body: Roboto Regular, 14px
• Small: Roboto Regular, 12px

Components:
• Material Design buttons
• Elevated cards
• Form inputs with validation
• Sortable tables
• Responsive modals
```

3. Format:
   - Font: Arial, 10pt
   - Color: Dark gray (#424242)
   - Background: Light gray (#F5F5F5)
   - Padding: 10px

---

## 📱 Step 5: Add Responsive Design Section

### Create Responsive Breakpoints Box

1. Click **Insert** → **Text Box**
2. Add the following content:

```
RESPONSIVE DESIGN

Mobile (<600px):
• Single column layout
• Hamburger menu
• Touch-friendly buttons
• Optimized images

Tablet (600-960px):
• Two column layout
• Collapsible sidebar
• Readable fonts
• Optimized spacing

Desktop (>960px):
• Three+ column layout
• Persistent sidebar
• Full navigation
• Optimized layout
```

3. Format:
   - Font: Arial, 10pt
   - Color: Dark gray (#424242)
   - Background: Light blue (#E3F2FD)
   - Padding: 10px

---

## 🎨 Step 6: Add Device Views

### Create Device Mockups

1. **Desktop View**:
   - Click **Insert** → **Shapes** → **Rectangle**
   - Draw a large rectangle (4" × 3")
   - Add border: 2pt, dark gray
   - Add text: "Desktop View (1920x1080)"
   - Add sample content inside

2. **Tablet View**:
   - Draw a medium rectangle (3" × 4")
   - Add border: 2pt, dark gray
   - Add text: "Tablet View (768x1024)"
   - Add sample content inside

3. **Mobile View**:
   - Draw a small rectangle (2" × 3.5")
   - Add border: 2pt, dark gray
   - Add text: "Mobile View (320x568)"
   - Add sample content inside

4. **Arrange side by side** to show responsive design

## 🎨 Step 9: Formatting Tips

### Colors to Use
- **Primary Blue**: #1976D2 (headers, highlights)
- **Light Blue**: #E3F2FD (backgrounds)
- **Success Green**: #4CAF50 (checkmarks)
- **Dark Gray**: #424242 (text)
- **Light Gray**: #F5F5F5 (backgrounds)

### Fonts
- **Headers**: Arial Bold, 24-28pt
- **Subheaders**: Arial Bold, 14-16pt
- **Body Text**: Arial Regular, 11-12pt
- **Code/Labels**: Courier New, 10pt

### Spacing
- Between sections: 0.25"
- Between screenshots: 0.15"
- Padding in boxes: 10px

## 💡 Pro Tips

4. **Add captions** - label each screenshot

6. **Highlight unique features** - medical vault, calendar, etc.
7. **Keep text readable** - use large fonts
8. **Use high-quality images** - avoid pixelated screenshots
